## Come iniziare

1. Clona il repository sul tuo computer:
   ```bash
   git clone https://github.com/tuonome/machine-learning-learning-hub.git
   ```
2. Apri la directory clonata con un editor per Jupyter Notebook (es. VsCode).
3. Happy learning!
